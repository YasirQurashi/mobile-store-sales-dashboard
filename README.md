# 📊 Mobile Store Sales Dashboard — Excel

## Overview
An interactive Excel dashboard analyzing mobile store sales 
across 4 cities in Pakistan with dynamic filters.

## 🔢 Key Metrics
- Total Sales: $28M
- Total Units Sold: 777
- Total Orders: 86

## 📈 Features
- Product-wise performance (Vivo, Redmi, Techno, Samsung, iPhone)
- City-wise sales breakdown (Islamabad, Karachi, Lahore, Sialkut)
- Month-wise revenue trends
- Top 5 Salesperson analysis
- Interactive slicers for Product, Region & Month

## 🛠️ Tools Used
- Microsoft Excel (Pivot Tables, Pivot Charts)
- Slicers for dynamic filtering
- Conditional Formatting
